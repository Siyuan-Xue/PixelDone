PixelDone v1.3.4 refines the main add-task control while keeping the editor flow unchanged.

Highlights:
- Moves the add-task action into the todo list panel as a bottom-centered floating control.
- Simplifies the add-task control to a square clay pixel button with a plain `+` mark.
- Keeps the new-task editor UI, todo storage, sorting, alarms, update checks, and haptics unchanged.
- Adds no new permissions, dependencies, or settings screen.

Install note: if you still have the old `com.codexue.pixeldone` prototype build, uninstall it before installing this private release.

The attached `PixelDone-1.3.4-release.apk` asset is the signed private release APK for direct installation and app-internal updates.
