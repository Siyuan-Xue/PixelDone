PixelDone v2.0.0 adds multiple local todo checklists while keeping the fast single-screen todo workflow.

Highlights:
- Adds multiple local todo checklists with a default `MAIN` list.
- Expands the top bar into a checklist switcher with per-list active and done counts.
- Shows the selected checklist name and matching active/done counts in the collapsed top bar.
- Keeps short press on `+` for new tasks and adds long press with haptic feedback for new checklists.
- Adds checklist editing with name input, save, cancel, and a separated destructive delete action.
- Migrates legacy single-list local todo data into the `MAIN` checklist.

Install note: if you still have the old `com.codexue.pixeldone` prototype build, uninstall it before installing this private release.

The attached `PixelDone-2.0.0-release.apk` asset is the signed private release APK for direct installation and app-internal updates.
