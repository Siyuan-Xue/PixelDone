PixelDone v2.7.3 is a signed formal release for dialog action alignment fixes.

Highlights:
- Keeps the delete confirmation dialog's `CLOSE` and `DELETE` actions on the same horizontal baseline.
- Uses one centered action row for the update prompt's `LATER` and `UPDATE` actions too.
- Confirms the update progress dialog has only one close action and does not share the paired-button offset issue.
- Confirms the image preview dialog title already centers its `CLOSE` control with the title row.

Install note: use the attached signed release APK, `PixelDone-2.7.3-release.apk`, for direct installation and in-app updates. This release installs as `com.milesxue.pixeldone`; uninstall incompatible debug/prototype builds only if Android reports a package or signature conflict.
